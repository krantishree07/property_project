# 🏡 Property Deal

**Property Deal** is a Basic JavaFX-based desktop application designed to simplify and manage the complete property buying and selling process. It provides a centralized platform where buyers, sellers, and administrators can efficiently handle client registration, property listings, buyer interests, property searches, secure deal completion, payment tracking, and analytical reports.

The application minimizes manual paperwork by maintaining a structured database of clients, properties, and transactions. It also includes interactive dashboards, graphical analysis, and Excel export functionality, making property management faster, more organized, and data-driven.

---

## ✨ Project Overview

Property Deal is developed to streamline real estate operations by integrating all essential modules into a single desktop application. The system allows administrators to manage customer information, property records, buyer preferences, payment status, and finalized deals with ease.

Using JavaFX for the user interface and MySQL as the backend database, the application provides secure data management, efficient record handling, powerful search capabilities, graphical analytics, and report generation for better business decision-making.

## 🔐 Login & Dashboard

The **Login** module provides secure user authentication, ensuring that only authorized users can access the Property Deal Management System. After successful login, users are redirected to the **Dashboard**, which serves as the central hub for managing all property-related operations. The dashboard offers quick navigation to client onboarding, property listing, buyer interest, property search, deal management, payment tracking, reports, and analytical modules.
